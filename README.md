I designed and implemented a multi-agent system for AI Infra covering both probing and auto-tuning functions. In Phase-1, I built custom CUDA probes integrated with Nsight Compute to perform local GPU hardware exploration and performance metric collection, producing structured reports that verified operator efficiency and hardware bottlenecks. Phase-2 extended the system with LLM and NCU driven strategies to iteratively evaluate and optimize CUDA kernels, automatically select optimal implementations and generate performance analysis reports. The system supports multi-round attempts, retry strategies,7 probe configurations,and kernel auto-tuning. This project integrates GPU profiling, CUDA optimization, multi-agent orchestration, and automated workflow design.
